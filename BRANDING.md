# Spratt Branding Guide

## Brand Colors

### Primary Colors
- **Spratt Blue**: `#00407A` (Primary brand color)
  - Used for: Main headings, primary buttons, links, accents
  - Hover state: `#003366` (darker shade)
  
- **Spratt Grey**: `#666666` (Secondary text color)
  - Used for: Secondary text, descriptions, metadata

### Background
- **White**: `#FFFFFF` (Primary background)
- **Light Grey**: `#F9FAFB` (Secondary backgrounds)

## Logo Usage

### Spratt Logo (`/Spratt_Logo.png`)
- Full logo with "SPRATT" and "INSURANCE BROKERS"
- Use on: Login page, headers, marketing materials
- Minimum size: 120px height

### Spratt Icon (`/Spratt_Icon.png`)
- Icon only version
- Use on: Favicon, small spaces, app icons
- Size: 32px - 64px typically

## Typography

### Headings
- Font: Sans-serif, bold, uppercase for "SPRATT"
- Color: Spratt Blue (`#00407A`)
- Size: 2xl - 3xl

### Body Text
- Font: System sans-serif
- Color: Spratt Grey (`#666666`) for secondary text
- Color: Dark grey/black for primary text

## Implementation

All brand colors are available in Tailwind as:
- `text-spratt-blue` / `bg-spratt-blue` / `border-spratt-blue`
- `text-spratt-grey` / `bg-spratt-grey` / `border-spratt-grey`
